# 本项目是关于spring-boot及其相关技术的使用

**项目介绍**
+ spring-boot-mybatis：Spring Boot与myBatis整合，采用的是mapper+xml的方式。
+ spring-boot: 以上所有组件的整合，前端采用layui库。
